Wynem Bot

Wynem is a multipurpose discord bot that has a vast array of features for all sorts of purposes. From Moderation, to Fun, to Image Editing!

Moderation

Wynem has full feature moderation with all the usual commands that you would expect from a moderation bot.

Fun

Wynem has lots of fun commands to use against your friends. You can do anything from snapping them, to throwing them off a cliff!

Utilities

Wynem has lots of utilities avaliable, such as setting per server reminders, so that you never forget to do anything!
